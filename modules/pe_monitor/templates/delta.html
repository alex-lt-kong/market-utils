Change over period

Forward P/E = Price ÷ Forward EPS, so the move decomposes left-to-right: (1 + Δ%) = (1 + Δ Price) / (1 + Δ Fwd EPS). Reading across: then → now, what each driver did (Δ Price, Δ Fwd EPS), then each driver's exact share — from Price + from EPS = Δ% (percentage points). P/E rises when price outpaces EPS, falls when EPS outpaces price.
